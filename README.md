# Gamer

Gamer is a user generated game review network built using the Ruby on Rails framework, HTML and CSS. Gamer includes features like game reviews, user comments, star ratings, and user authentication.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

What things you need to install the software and how to install them

Must have Ruby environment to run this application:

Can be found here: [Rbenv](https://github.com/rbenv/rbenv)

```
Ruby 2.3.1

Rails 5.0.3
```

### Installing

```
bundle install

bundle update

rake db:create:all

rake db:migrate
```

The application is now ready to use.

To run your server:

```
rails s
```

Open your web browser and navigate to

```
localhost:3030
```
